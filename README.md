# Agent Vision Coach

AI agent 视觉教练 — 基于多模态 Agent 的视觉质量评估与指导系统。

## 项目概述

利用 AI Agent 能力对视觉内容（图像、视频、UI 界面等）进行质量评估、缺陷检测、风格指导和一致性检查。

## 目录结构

```
.
├── src/               # 核心代码
├── config/            # 配置文件
├── tests/             # 测试
├── docs/              # 文档
└── examples/          # 使用示例
```
