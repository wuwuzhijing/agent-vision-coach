# Agent Vision Coach

**AI Agent 在视觉检测垂直领域的应用**

## 项目定位

将 AI Agent 的多模态理解、自主推理与工具调用能力落地到**工业/商业视觉检测**场景，实现：

- 图像/视频质量缺陷的自动化检测与分类
- 多视角、多模态的协同分析与决策
- 检测结果的智能研判、归因与报告生成
- 复杂检测流程的 Agentic 编排与自适应调度

目标是从单一的视觉算法模型，演进为具备**感知 - 推理 - 决策 - 行动**闭环的视觉检测智能体。

## 目录结构

```
.
├── src/               # 核心 Agent 逻辑与检测模块
├── config/            # 环境与模型配置
├── tests/             # 单元与集成测试
├── docs/              # 架构设计与技术文档
└── examples/          # 场景用例与 demo
```

## 场景方向

- **工业质检**：表面缺陷、尺寸测量、装配完整性
- **视频监控**：行为识别、异常事件检测
- **多模态协同**：图像 + 文本 + sensor 联合分析
- **检测流程编排**：多步骤质检的 Agent 自动调度
