# Agent Vision Coach

**AI Agent 在辅助驾驶视觉检测领域的应用**

## 项目定位

将 AI Agent 的多模态理解、自主推理与工具调用能力落地到**车载辅助驾驶**视觉检测场景，实现：

- **DMS（驾驶员监控系统）**：疲劳/分心/危险行为的智能研判
- **ADAS 感知**：目标检测、车道线、可行驶区域的 Agentic 分析与决策辅助
- **BSD（盲区检测）**：多视角融合的盲区风险判断
- **检测流程编排**：多个视觉算法的 Agent 协同调度、结果归因与报告生成

目标是从传统单一视觉算法模型，演进为具备**感知 - 推理 - 决策 - 行动**闭环的辅助驾驶视觉检测智能体。

## 目录结构

```
.
├── src/               # 核心 Agent 逻辑与检测模块
├── config/            # 环境与模型配置
├── tests/             # 单元与集成测试
├── docs/              # 架构设计与技术文档
└── examples/          # 场景用例与 demo
```

## 场景方向

- **DMS 智能研判**：结合上下文推理驾驶员状态，而非简单阈值触发
- **ADAS 感知增强**：多模型协同、跨帧分析、异常检测
- **BSD 融合决策**：多传感器 Agent 协同判断盲区风险
- **诊断与调试**：检测结果的智能归因、边缘 Case 分析与报告
